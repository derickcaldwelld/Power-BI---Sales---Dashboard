##Project Overview
This project analyzes a sample retail sales data using Power BI to identify trends in revenue, profit, and customer segments.

##Dataset
The dataset used in this project consists of 9,994 retail transactions.

##Key Insights from the dashboard

-The West Region shows the highest sales revenue with which we can see that the market demand is strong.
-The highest profit margin is generated by technology products.
-Tables, bookcases and supplies are loss making products.
-Copiers and phones are the products that generate the highest revenue.
